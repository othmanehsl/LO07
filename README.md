# LO07 - Projet BlaBlaCar

Application web de covoiturage en architecture MVC.
Université de Technologie de Troyes - LO07 Technologies du Web.

## Stack
- PHP (PDO)
- MySQL
- Bootstrap 5

## Installation
1. Importer `outil/blablacar2026.sql` dans MySQL
2. Configurer `app/controller/config.php`
3. Lancer via le serveur (XAMPP en local, dev-isi en production)
